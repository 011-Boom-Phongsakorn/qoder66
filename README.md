# โปรเจค Qoder

## คำอธิบาย

โปรเจคนี้เป็นที่เก็บไฟล์ข้อความภาษาไทยอย่างง่าย

## โครงสร้างไฟล์

- `hello.txt` - ไฟล์ข้อความคำทักทาย
- `world.txt` - ไฟล์ข้อความเพิ่มเติม
- `AGENTS.md` - เอกสารสำหรับ Qoder

## การใช้งาน

เปิดไฟล์ข้อความเพื่ออ่านเนื้อหาภาษาไทย

## ประวัติ

โปรเจคนี้สร้างขึ้นเพื่อทดสอบและเรียนรู้การใช้งาน Git และ GitHub
